# STEDTOP Pharmaceuticals — Project Requirements
# Static Website | HTML / CSS / Vanilla JavaScript
# ============================================================
# This is a purely static website with ZERO runtime dependencies.
# No npm install, no pip install, no build step required.
# All code runs directly in the browser.
# ============================================================


# ──────────────────────────────────────────────────────────────
# RUNTIME DEPENDENCIES (Browser — loaded via CDN)
# ──────────────────────────────────────────────────────────────

# Google Fonts — Loaded automatically via <link> in HTML <head>
# Playfair Display  — Display / heading serif font
# Inter             — Body / UI sans-serif font
# Source: https://fonts.googleapis.com/css2?family=Playfair+Display:ital,wght@0,400;0,500;0,600;0,700;1,400;1,600&family=Inter:wght@300;400;500;600&display=swap

# No JavaScript libraries or frameworks are used.
# No CSS frameworks (Bootstrap, Tailwind, etc.) are used.
# Everything is written in pure HTML5, CSS3, and Vanilla JS.


# ──────────────────────────────────────────────────────────────
# DEVELOPMENT TOOLS (Optional — for local development only)
# ──────────────────────────────────────────────────────────────

# Option A: VS Code + Live Server Extension (Recommended)
#   Editor  : Visual Studio Code >= 1.75.0
#             https://code.visualstudio.com/
#   Extension: Live Server by Ritwick Dey >= 5.7.9
#             https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer
#   Usage   : Right-click index.html → "Open with Live Server"
#   Port    : http://127.0.0.1:5500

# Option B: Node.js serve (if Node.js is installed)
#   Node.js  >= 14.0.0
#             https://nodejs.org/
#   Package  : serve (no install needed with npx)
#   Command  : npx serve .
#   Port    : http://localhost:3000

# Option C: Python HTTP Server (if Python is installed)
#   Python   >= 3.6
#             https://www.python.org/
#   Command  : python -m http.server 8080
#   Port    : http://localhost:8080

# Option D: Direct File Open (No tools needed)
#   Simply double-click index.html to open in any browser.
#   Note: Some browsers may block local font loading in this mode.
#         Use Live Server or Python server for best results.


# ──────────────────────────────────────────────────────────────
# BROWSER REQUIREMENTS
# ──────────────────────────────────────────────────────────────

# Google Chrome    >= 90
# Mozilla Firefox  >= 88
# Microsoft Edge   >= 90
# Apple Safari     >= 14
# Internet Explorer: NOT SUPPORTED
#
# Required browser APIs used in this project:
#   - IntersectionObserver  (scroll reveal animations)
#   - CSS Custom Properties (design tokens / variables)
#   - CSS Grid & Flexbox    (layout)
#   - CSS backdrop-filter   (frosted glass header)
#   - window.location.hash  (product filter routing)
#   - History API           (replaceState for filter URLs)
