# STEDTOP Pharmaceuticals — Website

> **Quality You Can Trust**

A professional multi-page static website for **STEDTOP Pharmaceuticals Pvt. Ltd.**, built using HTML, CSS, and Vanilla JavaScript.
The website showcases the company's therapeutic expertise, featured pharmaceutical products, quality commitment, and contact information in a modern responsive design.

## Pages

### Home (`index.html`)

The homepage introduces the company and highlights its expertise.

Features include:
- Animated hero section
- Company overview
- Therapeutic expertise section
- Featured pharmaceutical products
- Quality promise banner

---

### About (`about.html`)

The About page describes the company's mission and values.

Includes:
- Mission, Vision, Values
- Company story
- Therapeutic areas overview
- Quality commitment section

---

### Products (`products.html`)

Displays pharmaceutical products with a category filtering system.

Product categories include:
- Pain Management
- Neuro Care
- Gastro Care
- Wellness

Each product is displayed with:
- Product image
- Description
- Category tag

---

### Contact (`contacts.html`)

Provides contact details and a customer enquiry form.

Includes:
- Company email and address
- Business hours
- Contact form
- Enquiry type options

## How to Run

### Option 1 — Directly Open

Simply open the homepage file in any browser:

```
index.html
```

### Option 2 — VS Code Live Server

1. Install the **Live Server** Extension
2. Open the project folder in VS Code
3. Right-click `index.html`
4. Select **Open with Live Server**

### Option 3 — Node Local Server

```bash
npx serve .
```

### Option 4 — Python Server

```bash
python -m http.server 8080
```

Then open: [http://localhost:8080](http://localhost:8080)
